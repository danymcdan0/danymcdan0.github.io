This project was to create an ASP.NET Core Web API that could be consumed by an MVC Web Application while following Restful practices. This web application contains the use of Controllers to direct URI navigation and Domain Models/DTOs for displaying and saving data in an SQL database. It does this through the use of asynchronous programming utilizing the Repository Pattern. Further Azure was used to publish the site online.
